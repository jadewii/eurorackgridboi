THE HP SYSTEM EXPLAINED

HP = Horizontal Pitch = Module Width

In Eurorack, HP measures how wide a module is. Each HP equals 5.08mm (or about 1/5 inch). Our standard modules are 30HP wide.

U = Rack Unit = Vertical Height. Each U is 3 rack spaces tall. More U means more rows for modules!
